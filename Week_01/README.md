Stack：
    -Xss: 指定每个线程栈的字节数

HEAP:
    -Xmx: 指定最大堆内存
    -Xms: 指定堆内存的初始大小
    -Xmn: 年轻代内存大小

NON-HEAP:
    -XX: MaxMetaspaceSize, 设置Metaspace区内存大小，(JDK8+)一般不设置

OFF-HAEP:
    -XX: MaxDirectMemorySize, 设置最大堆外内存
